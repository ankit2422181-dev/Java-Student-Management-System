# Java Student Management System

A simple console-based Student Management System built using **Java** to manage student records efficiently.

## Features
- Add new student records
- View all student details
- Store student ID, name, and marks
- Uses OOP concepts
- Uses ArrayList for dynamic data storage
- User input handling using Scanner class

## Tech Stack
- Java
- OOP
- Collections Framework (ArrayList)

## Project Overview
This project was developed to strengthen core Java programming concepts such as object-oriented programming, collections, loops, conditional statements, and user interaction. It simulates a basic student record management system where users can add and view student information through a console interface.
